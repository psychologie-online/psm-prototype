prototype
=========

Pro potřeby prototypování. Pro ostrou verzi webu vytvoříme vlastní repozitář, který bude zaměřený na ladění už definitivních řešení.

+ [Icomoon](http://icomoon.io/app/#/select) – Ikonky pomocí @font-face
